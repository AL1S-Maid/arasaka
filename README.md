# Arasaka-dev

## Install pnpm

```sh
npm install -g bun
```

## Project Setup

```sh
bun install
```

```sh
bun dev
```

```sh
bun build
```

### Technology Stack

* [Vue](https://cn.vuejs.org/)
* [Vite](https://vitejs.cn/vite3-cn/)
* [Pinia](https://pinia.vuejs.org/zh/)
* [Threejs](https://threejs.org/)
* [Nestjs](https//nestjs.org/)
* [TypeScript](https://typescript.org/)
